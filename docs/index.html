Introduction

What is the External Resource Project?

An API which enables a developer to manage external resources in Hippo CMS.

We currently only support MediaMosa video platform integration.

Code base on GitHub

The code of this project is on GitHub at https://github.com/bloomreach-forge/external-resources/.
